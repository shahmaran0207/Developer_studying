File 객체의 메서드 delete()는 

1. 일반 파일이면 바로 지워짐
2.디렉토리는 비어있어야 지워짐
3. 따라서 빈 디렉토리가 아니면 아래 절차가 필요

ex) dir="E:/profile/1008"

1. 지정 폴더 안의 목록을 File[]로 반환
dir.listFiles();

2. 디렉토리 안에 파일이 여럿 존재할 수 있어 반복 수행
for(File file : dir.listFiles()){
	3. 각 파일에 대한 삭제를 실행
	file.delte();
}

4. 그 후 빈 디렉토리가 되면 바로 삭제 완료
dir.delete();
